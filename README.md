**Description**

CareConnect is a lightweight healthcare support platform for NGOs. 

It collects patient support requests, registers volunteers, and provides automated health guidance.



**Features**

\- Patient support request form

\- Volunteer registration

\- Automated issue categorization \& urgency detection

\- Healthcare FAQ chatbot

\- Firebase Firestore database



**Tech Stack**

\- HTML, CSS, JavaScript

\- Firebase Firestore

\- Hosted on Netlify



**AI / Automation Concept**

Keyword-based health issue classification and urgency detection to assist NGOs in faster triage.
